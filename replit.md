# My Website

## Overview
A simple website built with HTML, CSS, and JavaScript. This is a starter template that can be customized.

## Project Structure
- `index.html` - Main HTML file
- `style.css` - Stylesheet
- `script.js` - JavaScript functionality
- `server.py` - Python HTTP server to serve the static files

## Recent Changes
- 2025-10-10: Initial project setup with HTML, CSS, and JS files
- 2025-10-10: Added Python server for serving static files

## Features
- Responsive navigation
- Smooth scrolling
- Interactive button
- Clean, modern design

## Running the Project
The website is served using a Python HTTP server on port 5000.
